# Super Mario Sunshine - Tracker de pièces bleues
> **Un tracker web minimaliste, léger et modulaire pour suivre votre progression dans la quête des 240 pièces bleues de Super Mario Sunshine.**

Ce projet a été conçu pour offrir une interface claire et visuelle, palliant l'absence de suivi détaillé dans le jeu original.

---

## ✨ Fonctionnalités
* Suivi Progressif : Barres de progression globale et individuelles par zone.
* Organisation Modulaire : Chaque zone possède son propre fichier de données et son dossier d'images.
* Aide Visuelle : Support pour les captures d'écran individuelles par pièce et affichage de cartes multiples par zone.
* Visionneuse Intégrée : Cliquez sur n'importe quelle image ou carte pour l'ouvrir en grand et voir les détails.
* Sauvegarde Locale : Utilise le localStorage de votre navigateur pour conserver votre progression automatiquement.
* Export/Import : Sauvegardez votre progression dans un fichier .json pour la transférer ou la sécuriser.
* Design "Vanilla" : Développé sans frameworks lourds, utilisant uniquement HTML5, CSS3 et JavaScript pur.

---

## 🚀 Installation et Utilisation
Clonez ce dépôt, téléchargez les fichiers ou récupérez juste la dernière [Release](https://github.com/justeunedev/supermariosunshine-tracker/releases/latest) en zip.

Ouvrez simplement index.html dans votre navigateur préféré (une fois les fichiers dézippés). Aucun serveur n'est requis, tout tourne en local.

Virustotal de la dernière release : [Rapport VirusTotal de la 1.0](https://www.virustotal.com/gui/file/fcf9ab4eeaa4c911dd97b5c825b914fd30eb62a2245bd20b37eb4a16020de1bd?nocache=1)
